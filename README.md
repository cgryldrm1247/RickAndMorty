# RickAndMorty 

Bu proje KONUŞARAK ÖĞREN firmasının verdiği case sonucunda yazılmıştır
Bir Tanıtım uygulamasıdır, kullanıcı login olmadan istediği gibi uygulama içinde gezinebilir.
Eğer bir karakteri favoriye almak isterse maksimum 10 adet karekteri favoriye alabilir.
Bütün bölümleri görebildiği paginationlanmış içerikleri görebilir.
Favori karakterlerini görebileceği ekran ayarlanmıştır.

## Kurulum

1. İlk olarak, projeyi klonlayın.
2. Daha sonra, projenin klasörüne gidin ve şu komutu çalıştırın: ilk önce Expo'yu kurun[	
npm install -g expo-cli] sonra npm install atın :[npm install] ve sonra çalıştırın: [npm start]
